LZ77 compression

Java :: LZ77 compression

This project was something that I completed in my second year of college. In my maths module, we learned about different types of compression/decompression algorithms, in particular LZ77 & LZW. I tasked myself with replicating the LZ77 algorithm in Java. The project in its current state is limited. The user has to manually type out and enter the character stream which represents the image, which doesn't make it feasible for larger images. The program outputs an encoding/decoding table which shows how each pixel in the image compressed/decompressed, along with the reduction achieved at the end. Examples of 8x8 images can be found in the README on my github.

Tools used: Java, JetBrains Intellij

Platforms: Cross

LZW compression

Java :: LZW compression

Similar to the LZ77 project, this project was something I completed in my second year of college. The LZW algorithm was the other compression algorithm we learned about, and for the same reason as the LZ77 project, I tasked myself with replicating the algorithm in Java. Similarly, this project is quite limited. The user has to manually type out and enter the character stream which represents the image, however the program does output the root dictionary, encoding table, extended dictionary, code-stream, and the reduction achieved. An example of a 16x16 image being compressed can be found in the README on my github.

Tools used: Java, JetBrains Intellij

Platforms: Cross

llog

C++ :: Logging Library

llog is an easy to use, customizable C++ logging library. The inspiration for this project came during my time on my internship with ericsson. I was exposed to the Java logging library log4j, from the beginning and used it through out my entire time there. As C++ is my preferred language, I tasked myself with developing a simple logger, based on the design of log4j, that I could use for my own future C++ projects.

The library provies a simple API for creating log messages at different log levels, as well as settings custom formats and colors and enabling/disabling log levels. Instructions on how to install and use the library can be found in the README on my github.

Tools used: C++, Catch2, JetBrains CLion

Platforms: macOS (Clang 3.5+)

dt

C++ :: Date and Time Library

dt is an easy to use C++ date and time library. This project was the product of some spare time I had towards the end of my internship at Ericsson. There was no particular usecase that I needed it for, however I wanted to take it as a learning experience in C++ library development.

The library offers many features such as date and time manipulation, calendar calculations, stopwatch functionality, handling formatted dates and exception handling for invalid dates and times. Instructions on how to install and use the library can be found in the README on my github.

Tools used: C++, Googletest, JetBrains CLion

Platforms: macOS (Clang 3.5+)

space invaders

C++ :: space invaders

This project was something I started shortly after I discovered the graphics library SFML. I had wanted to make a game with graphics entirely in C++ and i chose to try and make space invaders. In the games current state, the player can, move left/right, shoot, take damage, pick up hearts to increase their health, pick up shields to give extra protection. Enemies shoot randomly and explode when hit by a player bullet. A short demo of the game can be viewed in the README on my github.

NOTE: Currently this project is not finished. The game is only playable until the player kills all the enemies or the enemies kill/pass the player. I plan to revist this in the future.

Tools used: C++, JetBrains CLion, SFML

Platforms: macOS (Clang 3.5+)

Dependencies: SFML v2.5
